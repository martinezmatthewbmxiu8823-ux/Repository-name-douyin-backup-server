# 抖音罗盘全量数据自动备份系统 v2.0

## 🎯 更新内容

现在支持**所有榜单类型**的自动备份！

### 支持的榜单

- 🎬 **短视频榜** - 热门短视频排行
- 📦 **商品榜** - 热销商品排行
- 📺 **直播榜** - 直播间排行
- 👤 **达人榜** - 达人/创作者排行

---

## 📊 新增API接口

### 1. 获取所有榜单类型

```bash
GET http://localhost:3000/api/ranks
```

响应：
```json
{
  "success": true,
  "ranks": [
    {"key": "video", "name": "短视频榜", "collection": "videos"},
    {"key": "product", "name": "商品榜", "collection": "products"},
    {"key": "live", "name": "直播榜", "collection": "lives"},
    {"key": "creator", "name": "达人榜", "collection": "creators"}
  ]
}
```

### 2. 获取指定榜单的数据

```bash
# 短视频榜
GET http://localhost:3000/api/video/2026-07-04

# 商品榜
GET http://localhost:3000/api/product/2026-07-04

# 直播榜
GET http://localhost:3000/api/live/2026-07-04

# 达人榜
GET http://localhost:3000/api/creator/2026-07-04
```

### 3. 触发全量备份

```bash
POST http://localhost:3000/api/backup/trigger
```

响应：
```json
{
  "success": true,
  "message": "全量备份任务已触发",
  "results": {
    "video": {"success": true, "count": 50},
    "product": {"success": true, "count": 100},
    "live": {"success": true, "count": 30},
    "creator": {"success": true, "count": 50}
  }
}
```

---

## 🚀 自动备份

系统会在**每天凌晨2点**自动备份所有榜单：

1. 短视频榜
2. 商品榜
3. 直播榜
4. 达人榜

每个榜单间隔3秒，避免请求过快。

---

## 📁 数据存储结构

MongoDB中会创建4个集合：

```
douyin_backup
├── videos      # 短视频榜数据
├── products    # 商品榜数据
├── lives       # 直播榜数据
└── creators    # 达人榜数据
```

每个集合的数据结构：
```json
{
  "date": "2026-07-04",
```

## 💡 使用示例

### 查询所有榜单的今日数据

```javascript
const today = '2026-07-04';

// 并行查询
const [videos, products, lives, creators] = await Promise.all([
  fetch(`http://localhost:3000/api/video/${today}`).then(r => r.json()),
  fetch(`http://localhost:3000/api/product/${today}`).then(r => r.json()),
  fetch(`http://localhost:3000/api/live/${today}`).then(r => r.json()),
  fetch(`http://localhost:3000/api/creator/${today}`).then(r => r.json())
]);

console.log('短视频:', videos.data.count, '条');
console.log('商品:', products.data.count, '条');
console.log('直播:', lives.data.count, '条');
console.log('达人:', creators.data.count, '条');
```

### 手动触发备份

```javascript
fetch('http://localhost:3000/api/backup/trigger', {
  method: 'POST'
})
.then(r => r.json())
.then(result => {
  console.log('备份结果:', result.results);
});
```
